
# 🧾 Générateur de mails administratifs (Micro SaaS)

Ce projet est un micro SaaS Flask qui génère des mails administratifs automatiquement.

---

## 🚀 Fonctionnalités

- Génération de mails administratifs
- Templates pour plusieurs services :
  - Catastro
  - Avocat
  - Banque
- Interface simple et rapide
- Bouton copier le mail
- Version web déployable (Railway)

---

## 🛠️ Technologies utilisées

- Python 3
- Flask
- HTML / CSS
- Gunicorn (production)

---

## 📦 Installation locale

bash
pip install -r requirements.txt
python app.py
